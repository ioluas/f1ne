# f1ne

F1 racing application for viewing F1 related data.

---

### Built using:

- [fyne framework](https://fyne.io/)
- [Ergast API](https://ergast.com/mrd/)
- [logrus logger](https://github.com/sirupsen/logrus)
- [bitcask datastore](https://git.mills.io/prologic/bitcask)
- [Race icons created by Freepik - Flaticon](https://www.flaticon.com/free-icons/race)
